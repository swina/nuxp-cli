# Usage
<!-- usage -->
```sh-session
$ npm install -g nuxp-cli
$ nuxp-cli COMMAND
running command...
$ nuxp-cli (-v|--version|version)
nuxp-cli/0.0.0 win32-x64 node-v12.18.3
$ nuxp-cli --help [COMMAND]
USAGE
  $ nuxp-cli COMMAND
...
```
